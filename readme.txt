The plugin allows you to place a table from a CSV file on your site.

== Description ==

Plugin CSV Sumotto help WordPress users to place on your pages table from files CSV. Making the table is done with an editor, it is possible to choose the background color, font, font color and size, specify the scope and size of the cells, is also available merging cells. Content is taken directly from the file csv, so any changes to the file to be displayed in the table on your site.

== Installation ==

1. Download the zipped plugin file to your local machine.
2. Unzip the file.
3. Upload the "csv_sumotto" folder to the "/wp-content/plugins/" directory.
4. Activate the plugin through the "Plugins" menu in WordPress.
5.Download the file to a folder 
6.Navigate to Tables -> Add New, create a table.
7.Navigate to Tables, select the table you want, edit, and save.
8.Go to the editor pages / notes, paste Shortcode in a place where you want to display a table.

=shortcode=

[csv id='id_table']
Shortcode can be found in menu "all tables" in the column shortcode desired table.
